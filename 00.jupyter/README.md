# Instalación del kernel de Jupyter para Java
https://github.com/SpencerPark/IJava

## Pasos a dar:

1. Descargar el proyecto:
```bash
git clone https://github.com/SpencerPark/IJava.git
cd IJava/
```

2. Instalación:
- Unix: 
```bash
./gradlew installKernel
```
- Windows: 
```bash
gradlew installKernel
```
